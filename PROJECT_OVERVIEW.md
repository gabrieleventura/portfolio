# Project Overview — Shattered Minds (Cervelli Esplosi)

Shattered Minds is a self-hosted bilingual website project focused on clear communication and accessibility.  
It is maintained as a long-term, real-world environment to practise production-minded Linux operations and disciplined documentation.

## My Role
Solely responsible for hosting and operations end-to-end, including initial deployment, ongoing maintenance, and reliability improvements.

## What This Project Demonstrates
- Linux system administration fundamentals applied to a live service
- Security-minded operational practice (baseline-first approach, least privilege, controlled changes)
- Troubleshooting through structured analysis and root-cause reasoning
- Documentation-driven workflows (runbooks, checklists, clear operational notes)
- Ongoing continuous improvement and maintainability over time

> Note: This portfolio intentionally avoids publishing configuration details, security rules, or system-specific implementation information.
